ChatGPT 接口

完全兼容 OpenAI 官方接口格式，支持所有 GPT 模型。

POST /v1/chat/completions 创建聊天补全（支持流式）
POST /v1/audio/transcriptions 音频转文字 (Whisper)
POST /v1/audio/speech 文字转语音 (TTS)
POST /v1/embeddings 创建文本嵌入向量

Python 配置示例

使用 OpenAI 官方 Python SDK，只需修改 base_url 即可。

import openai

# 配置小不丢API
openai.api_base = "https://api.augmentproxy.com/v1"
openai.api_key = "your-api-key"

# 调用接口
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

print(response.choices[0].message.content)
